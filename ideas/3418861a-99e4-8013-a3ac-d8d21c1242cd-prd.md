## Problem
The current Fullstack mascot is an axolotl, which Chase finds uninspiring and difficult to work with. It doesn't fit the app's identity — Fullstack is framed around a video game progression metaphor (levels, XP, skill trees), and the mascot should reinforce that, not contradict it.

## Goal
Replace the axolotl with a video game character mascot that better aligns with Fullstack's game-feel aesthetic and is easier to iterate on visually.

## Users
Chase, as the primary (and likely sole) Fullstack user. The mascot is a brand decision, but it also shapes first impressions in onboarding and the emotional tone of the product overall.

## Requirements

**Must have**
- A new mascot design concept defined (character archetype, style, personality — pixel art, retro RPG, platformer hero, etc.)
- Existing axolotl references removed or replaced across the UI
- Mascot appears in at least one active surface (e.g., level-up screen, onboarding, empty state)

**Should have**
- The mascot has a small set of states or poses (idle, celebrating, struggling/losing) for use across different UI moments
- The mascot's visual style is consistent with the existing Fullstack UI color palette and design system
- Character has a name or identifier for reference in code and copy

**Nice to have**
- Simple sprite-style animation for level-up or achievement moments
- Mascot reacts to quiz performance (happy on streak, deflated on wrong answers)

## Out of scope
- Full character animation system
- Multiple mascot characters or unlockable skins in v1
- External design contractor or asset purchase — this is assumed to be designed in-house or AI-generated

## Open questions
- What is the visual style of Fullstack today — pixel art, flat, illustrated? The mascot needs to match.
- Are axolotl assets custom-made or sourced? If custom, what's the replacement path?
- Does the mascot appear in any external-facing materials (app store, marketing) or only in-product?

## Decision log
- Axolotl dropped as mascot — noted as "dumb and hard" (difficulty of illustrating/animating likely a factor)
- Video game character direction chosen to reinforce the core game-progression metaphor already embedded in Fullstack's design
